# 47. AI-PM Task Bot (【AI-PM】Notion×Slack連携のプロジェクト進行管理ボット)

⭐ **If you find this workflow helpful, please give it a Star! (役に立ったらぜひスターをお願いします！)** ⭐

## 🚀 概要 (Overview)
「AIに作業を指示する」時代から、**「AIにプロジェクトマネージャー（PM）を任せて、人間が指示を受ける」**時代へ。
毎朝、Notionのタスク管理データベースをAI（Gemini）が読み込み、期限切れのタスクや今日やるべき最優先タスクを整理して、チームのチャット（SlackやChatwork）へ自動でブリーフィング（指示出し）を行う画期的なワークフローです。

このボットを導入することで、人間は「今日何から手をつければいいんだっけ…」と悩む時間をゼロにし、朝イチから迷わず実作業に没頭できるようになります。

## ✨ 特徴 (Features)
* **逆転の発想（AIからの指示出し）:** 人間がプロンプトを考えるのではなく、AIが自律的に状況を把握して人間にアクションを促します。
* **文脈を理解した優先順位付け:** 単なる「タスクの羅列」ではなく、期日と現在のステータスをAIが読み解き、「これはヤバいので最優先で！」といったPM視点でのトリアージ（優先順位付け）を行います。
* **モチベーション管理:** 少し元気なトーンで指示を出すようプロンプトを設計しており、チームの朝の活力を高めます。

## ⚙️ 使用しているノード (Nodes Used)
1. **Schedule Trigger:** 毎朝9時など、チームの始業時間に合わせて起動。
2. **Set (Edit Fields):** テスト用のNotionタスク一覧を生成（※実運用時はNotionノードに置換します）。
3. **Google Gemini:** タスク一覧を読み込み、PMとしてチームへの指示（JSON）を生成。
4. **Code (JSONパース):** Geminiの出力から純粋なJSONオブジェクトを抽出。
5. **Chatwork / Slack:** 生成されたブリーフィングテキストをチームのチャンネルへ通知。

## 🛠️ 使い方と実運用への移行手順 (How to Use & Setup)

### 💡 まずはテスト実行で「AIのPM力」を体感する
本ワークフローには、インポートしてすぐに試せるよう、ダミーのタスク一覧（期日遅れを含む）を出力するSetノードが組み込まれています。
まずはそのまま「Test workflow」を実行し、AIが見事に優先順位をつけて指示を出してくる感動を味わってください。

### ⚠️ 実運用環境（本物のNotion連携）への移行ステップ
実際の業務で使用する際は、ダミーの「Setノード」を削除し、以下の手順で本物のNotionと連携させます。

1. **Notionノードの追加:**
   `Notion` ノードを追加し、「Database Page」>「Get Many」を選択します。
   対象となるタスク管理データベースの `Database ID`（URLの32文字の英数字）を入力します。
   ※この時、Notion側でn8nアプリのアクセス許可（コネクトの追加）を忘れずに行ってください。
2. **完了タスクの除外（Filter）:**
   終わったタスクまでAIに読ませないよう、Notionノードのオプションで「ステータスが完了と一致しない」などのフィルターをかけます。
3. **Codeノードによるテキスト結合（重要）:**
   Notionから取得した複数のタスクアイテムを、AIが読める「1つの長いテキスト」にまとめる必要があります。NotionノードとGeminiノードの間に `Code` ノードを挟み、以下のようなJavaScriptでテキスト化してください。

   ```javascript
   let taskText = "【現在のタスク一覧】\n";
   const items = $input.all();
   items.forEach(item => {
     // ※プロパティ名はご自身のNotionに合わせて変更してください
     const name = item.json["タスク名"] || "不明";
     const deadline = item.json["期日"] || "なし";
     taskText += `・${name} (期日: ${deadline})\n`;
   });
   return [{ json: { notion_tasks: taskText } }];
   ```
