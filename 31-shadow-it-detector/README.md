# 31. Shadow IT Detector (シャドーIT・野良SaaS自動検知ボット)

## 🚀 概要
クレジットカード明細から、会社が許可していないクラウドサービスやソフトウェア（シャドーIT）の利用を自動検知し、情報漏洩リスクを未然に防ぐセキュリティ監査ボットです。
Webフォームから明細CSVをアップロードすると、Geminiが情シス担当マネージャーとして経費データを分析。スタバや交通費などの非IT経費は無視しつつ、NotionやChatGPTなどの未承認SaaS利用をリストアップして警告レポートをメール送信します。

## ✨ 特徴
* **高精度な仕分け:** ホワイトリスト（許可済みツール）と照合し、未知のSaaS名であってもAIが「ITサービスである」と文脈から推測して検知します。
* **コンプライアンス強化:** 単なるリストアップに留まらず、「なぜそのツールを勝手に使うと危険なのか（データ漏洩、退職者アカウント管理など）」という具体的なリスクと対応方針までレポート化します。
* **SECURITY ACTION対応:** IPAなどのセキュリティ認証取得に向けた「情報資産管理・アクセス制御」の具体的な運用実績として活用できます。
* **CSVインポート対応:** 実運用を想定し、Webフォームからのファイルアップロード機能とCSVパース処理を実装しています。

## ⚙️ 使用しているノード
* **Form Trigger:** 経費データ監査ポータル（Webフォーム）を提供し、CSVファイルを受け付けます。
* **Spreadsheet File:** アップロードされたCSVファイルを読み込み、JSONデータに変換します。
* **Code:** 抽出したデータをGeminiが読みやすい1つのテキストに結合します。
* **Google Gemini:** 経費データを分析し、シャドーITの疑いがある決済と具体的な推奨アクションを出力します。
* **Gmail:** Geminiの監査レポートを管理者へ送信します。
* **Respond to Webhook:** フォーム送信後のブラウザ画面に完了メッセージを表示します。
