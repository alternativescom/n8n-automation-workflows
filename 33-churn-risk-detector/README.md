# 33. Churn Risk & Complaint Detector (お怒り・解約リスク自動検知ボット)

## 🚀 概要
顧客からの問い合わせメールをAIがリアルタイムで監視し、「怒りの感情」や「他社への乗り換え（解約）を匂わせるワード」を検知した場合にのみ、CS（カスタマーサクセス）マネージャーや営業チームへ緊急アラートを飛ばす早期警報システムです。
クレーム対応における初動の遅れを防ぎ、顧客満足度の低下や解約（チャーン）を未然に防ぎます。

## ✨ 特徴
* **リアルタイム監視:** Gmailトリガーを使用し、サポート宛のメールを受信した瞬間にAIが文面を分析します。
* **ベテランCSの視点:** 単なるキーワード検知ではなく、Geminiが「顧客がなぜ怒っているのか（ペインポイント）」を読み解き、二次クレームを防ぐための具体的な「初動対応3ステップ」まで指示を出します。
* **HTMLメール対応:** AIが生成したMarkdownレポートをn8n上で綺麗なHTMLに変換し、視認性の高いアラートメールとして関係者に配信します。

## ⚙️ 使用しているノード
* **Gmail Trigger:** 指定したラベル（INBOX等）に新しいメールが届いたことをトリガーとしてワークフローを起動します。
* **Google Gemini:** メールの件名と本文から、顧客の感情レベルと解約リスクを分析します。
* **Markdown:** Geminiが出力したMarkdown形式のテキストを、メールで読みやすいHTML形式に変換します。
* **Gmail:** 変換されたHTMLレポートを、マネージャーや担当チームのメールアドレスへ送信します。
