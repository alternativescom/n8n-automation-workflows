# 50. Oshi-Katsu Ticket Alert Bot (【推し活ハック】HANAイベント情報＆チケット特急アラート)

## 🚀 概要 (Overview)
**【祝・n8nワークフロー100本ノック（第50弾達成記念）🎉】**
自動化の力は、ビジネスだけでなく「個人の情熱」を加速させるためにも使えます。
このワークフローは、大好きなアーティスト（推し）のライブ開催情報やチケットの先行抽選など、「絶対に見逃してはいけない情報」を自動で収集し、AIに「今すぐ申し込むべき緊急メッセージ」を作らせて、スマホへダイレクトに通知するボットです。

公式発表の無機質な情報だけでなく、「受付期間が3日しかない！」といったファンのSNSでの焦り（コンテキスト）も一緒にAIに読み込ませることで、スマホの画面を見た瞬間にタップしてしまう強烈なアラートを生成します。

## ✨ 特徴 (Features)
* **情報の掛け合わせ:** 公式サイトのニュースと、ファンのSNSの反応を同時に処理し、状況の「緊急度」を正確に把握します。
* **行動を促すAIプロンプト:** AIを「熱狂的なアシスタント」として設定することで、「〇日から受付開始です」という単なる通知ではなく、「野田さん緊急！今すぐエントリーを！」という行動直結型のメッセージを生成します。
* **スマホ直撃のプッシュ通知:** メールなどの埋もれやすいツールではなく、SlackのDMや各種メッセージアプリを使って確実にスマホを震わせます。

## ⚙️ 使用しているノード (Nodes Used)
* **Manual Trigger / Set:** テスト実行・ダミーデータ生成用（※実運用時は削除します）
* **Google Gemini:** 複数情報の要約と、緊急メッセージの生成
* **Code:** JSONのパース処理
* **Slack:** 自分宛てのダイレクトメッセージ（DM）送信

## 🛠️ 実運用環境への移行手順（本気の推し活設定）

インポート直後のワークフローは「テスト用」にダミーデータがセットされています。
実際の情報収集ボットとして稼働させるには、以下の手順でノードを置き換えてください。

### Step 1: トリガー（起点）を「定期実行」に変更する
一番左にある `Manual Trigger` ノードを削除し、代わりに `Schedule Trigger` ノードを配置します。
チケット情報はスピードが命なので、設定は「Every 3 Hours（3時間ごと）」や「Every Day at 12:00（毎日お昼）」などに設定します。

### Step 2: 公式サイトやSNSの情報を自動取得する
テスト用の `Set (ダミー情報取得)` ノードを**削除**し、実際に情報を取得するノードに置き換えます。
* **公式ニュースの取得:**
  `HTTP Request` ノードや `RSS Read` ノードを使い、アーティストの公式サイトのNEWSページや、チケットサイトの情報を取得します。（※HTMLからテキストを抽出する場合は `HTML Extract` ノードを併用します）。
* **SNSの反応の取得:**
  現在のX（Twitter）はAPI制限が厳しいため、Yahoo!リアルタイム検索のRSSを活用したり、特定のハッシュタグを収集する外部サービスを `HTTP Request` で叩く方法がおすすめです。

### Step 3: 取得したデータをGeminiに紐付ける
情報を取得するノードと `Gemini` ノードを線で繋ぎます。
Geminiノードを開き、プロパティの「Messages」内にある `{{ $json.official_news }}` などの部分を消し、左側の入力データ一覧から、**Step 2で実際に取得したテキストデータ（変数）**をドラッグ＆ドロップして紐付け直します。

## 📱 Slack以外の通知ツールを使いたい場合

「Slackは仕事で使っているから、推し活の通知は別のアプリで受け取りたい」という場合は、最後尾の `Slack` ノードを削除し、以下のいずれかのノードに置き換えてください。

* **💬 LINEで受け取る:**
  `LINE` ノードを追加します。事前にLINE Developersで「Messaging API」のチャネルを作成し、アクセストークンを取得して設定してください。ご自身のLINEに直接メッセージが届きます。
* **🎮 Discordで受け取る:**
  `Discord` ノードを追加します（または `Webhook` ノードを使用）。ご自身のDiscordサーバーに専用チャンネルを作り、Webhook URLを発行して設定画面に貼り付けるだけで、超高速なプッシュ通知が実現します。
* **✈️ Telegramで受け取る（おすすめ）:**
  `Telegram` ノードを追加します。Telegramアプリ内で「BotFather」に話しかけてボットを作成し、トークンを取得するだけで簡単に個人用通知ボットが作れます。海外の自動化エンジニアに最も人気のある通知ルートです。
