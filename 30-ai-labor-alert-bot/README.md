# 30. AI Labor Alert Bot (深夜チャット・労務アラートボット)

## 🚀 概要
フルリモート環境下で見逃されがちな「隠れ残業」や「メンタル不調のサイン」をAIが検知する労務管理ボットです。
深夜や休日のチャットログ（Slack/Teams等）をn8nが定期取得し、産業カウンセラー資格を持つ「ベテラン労務マネージャー」としてGeminiが感情や疲労度を分析します。リスクが高いと判定された場合のみ、管理者へ具体的な声掛け（1on1等）の推奨アクションを通知します。

## ✨ 特徴
* **メンタルケア重視:** 単なる労働時間の監視ではなく、テキストのトーンから「焦り・疲労・SOS」を読み取るケア目的のツールです。
* **バッチ処理によるコスト削減:** メッセージのたびにAIを動かすのではなく、毎朝のスケジュール実行で前日分のログを一括処理するため、APIの消費を抑えられます。
* **柔軟な通知先:** 管理者のSlackチャンネルや、人事担当者のGmailなど、要件に合わせて通知先を簡単に切り替え可能です。

## ⚙️ 使用しているノード
* **Schedule Trigger:** 毎朝8時など、決まった時間にワークフローを起動。
* **Code (Mock):** ※本プロトタイプでは、テスト用の深夜チャットログを擬似的に生成しています。実運用時はSlack等の履歴取得ノードに差し替えます。
* **Google Gemini:** チャットログを読み込み、労務リスクのレベルと具体的な推奨アクションをテキストで出力。
* **Gmail (または Slack):** Geminiの分析レポートを管理者へ送信。
