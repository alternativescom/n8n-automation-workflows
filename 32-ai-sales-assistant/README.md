# 32. AI Sales Assistant (インサイドセールス向け・商談準備ボット)

## 🚀 概要
自社サイトにお問い合わせ（リード獲得）があった瞬間、AIが見込み客の企業サイトを自動でスクレイピング・分析し、営業担当者へ「1分で読める商談準備シート」を届けるワークフローです。
営業担当が「ググって会社概要を読む時間」をゼロにし、初回の商談から精度の高い「提案の切り口」や仮説を持ってアプローチできるため、受注率の向上と営業スキルの属人化解消に貢献します。

## ✨ 特徴
* **自動スクレイピング＆抽出:** フォームに入力されたURLへ即座にアクセスし、HTMLのbody要素からテキストデータのみを抽出することで、トークン消費を抑えつつ高速に分析します。
* **質の高い仮説構築:** 企業概要の要約だけでなく、問い合わせ内容と掛け合わせて「推測される真の課題」や「商談での提案の切り口（3つ）」など、実務に直結するインサイトをAIが生成します。
* **「攻め」のDX:** 経理や労務といったバックオフィス領域にとどまらず、売上に直結する営業部門の生産性を劇的に高めるツールです。

## ⚙️ 使用しているノード
* **Form Trigger:** お問い合わせを受け付けるWebフォーム（企業名、URL、相談内容）。
* **HTTP Request:** フォームに入力された対象企業のURLへアクセスし、Webページ（HTML）を取得します。
* **HTML Extract:** 取得したHTMLから不要なコードを除外するため、bodyタグ内のテキストだけを抽出します。
* **Google Gemini:** 企業のWebサイト情報と相談内容を分析し、インサイドセールス用の商談準備シートを生成します。
* **Gmail:** 完成したレポートを営業担当者のメールアドレスへ送信します。
* **Respond to Webhook:** フォーム送信後のブラウザ画面に受付完了メッセージを表示します。
