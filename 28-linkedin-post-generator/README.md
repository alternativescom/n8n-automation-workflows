# 28. LinkedIn Post Generator (BtoB向け LinkedIn投稿・自動生成フォーム)

## 🚀 概要
n8nの「Form Trigger」と生成AI（Gemini）を組み合わせた、社内の非エンジニアでも簡単に使えるSNS投稿文の自動生成ツールです。
n8nのキャンバス（編集画面）を開くことなく、発行されたWebフォーム（URL）から「ターゲット層」「参考記事のURL」「伝えたいコアメッセージ」を入力するだけで、AIがビジネスパーソン向けの魅力的なLinkedIn投稿文を作成し、画面上に返してくれます。

## ✨ 特徴
* **直感的なUI:** HTMLやサーバー構築不要で、n8n上で入力フォームが完結します。
* **BtoB特化のプロンプト:** 単なる文章作成ではなく、ターゲット層への共感フックや適切なハッシュタグを含んだビジネス向けの構成で出力します。
* **脱・属人化:** 営業担当や経営陣が各自のスマホやPCからフォームに入力するだけで、高品質な発信を継続できるようになります。

## ⚙️ 使用しているノード
* **n8n Form Trigger:** ユーザー入力用のWebフォーム（ドロップダウン、テキスト入力等）を提供。
* **Gemini (Google AI):** 入力情報に基づく投稿文の生成（プロンプト処理）。
* **Respond to Webhook:** 生成されたテキストを、入力者のブラウザ画面にそのままテキストとして表示。

## 🚀 導入手順
1. 本リポジトリの `workflow.json` をn8nにインポートします。
2. `Gemini` ノードにGoogle AI StudioのAPIキー（認証情報）を設定します。
3. `Gemini` ノードのプロンプト内にある前提条件を、自社の事業内容（例：ITコンサルティング、テレワーク支援など）に合わせて微調整するとさらに精度が上がります。
4. `Form Trigger` ノードを開き、「Test URL」または「Production URL」をクリックしてブラウザでフォームを開きます。
5. フォームにテスト入力を行い、送信（Submit）して結果が表示されれば成功です。社内メンバーにはこのURLを共有してください。

## 💡 カスタマイズ（出力先の変更）
デフォルトでは「ブラウザ画面に結果を返す（Respond to Webhook）」設定になっていますが、自社の運用に合わせて最後のノードを差し替えることが可能です。

* **Slack / Teams連携:** `Slack` ノード等に差し替え、生成結果を指定のチャンネルや担当者へのDMとして自動送信する。
* **Notion連携:** `Notion` ノードに差し替え、社内の「SNS投稿カレンダー」データベースにドラフト（下書き）として自動保存する。
