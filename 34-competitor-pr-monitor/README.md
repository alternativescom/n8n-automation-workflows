# 34. Competitor PR Monitor (競合PR監視＆対抗策提案ボット)

## 🚀 概要
PR TIMESなどのRSSフィードを毎日自動で巡回し、競合他社が新しいプレスリリースを発信した際に、AIが「敏腕マーケティング責任者（CMO）」としてその内容を分析・評価するツールです。
単なる「ニュースの通知」に留まらず、そのリリースが自社に与える脅威レベルを判定し、「明日からマーケティング・営業チームが実行できる対抗策」までをセットにして、視認性の高いHTMLメールで経営層・マーケティングチームへ届けます。

## ✨ 特徴
* **情報収集の完全自動化:** 毎朝スケジュール実行でRSSを取得するため、広報やマーケティング担当者が毎日ニュースサイトを巡回する手間がゼロになります。
* **CMO視点の戦略提案:** ニュースを要約するだけでなく「自社への影響分析」と「3つの対抗策」をAIに思考させることで、ニュースを見たあとの初動スピードを劇的に高めます。
* **見やすいHTMLレポート出力:** AIが生成したリッチなMarkdownテキストをn8n上でHTMLに変換し、そのままGmailで関係者に配信します。

## ⚙️ 使用しているノード
* **Schedule Trigger:** 毎朝9時など、決まった時間にワークフローを起動します。
* **RSS Read:** PR TIMES等、指定した競合他社のRSSフィードURLから最新の配信データを取得します。
* **Google Gemini:** リリースの内容を分析し、脅威レベルと自社への影響、および対抗策のアイデアを生成します。
* **Markdown:** Geminiが出力したレポートを読みやすいHTML形式に変換します。
* **Gmail:** 変換されたHTMLレポートを、マーケティングチームや経営層のメールアドレスへ送信します。
