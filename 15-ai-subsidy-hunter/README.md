# 💰 n8n-workflow #15: AI Subsidy Hunter
> **「知っている人だけが得をする」を終わらせる。AIがあなたの会社に最適な補助金を24時間自動発掘。**

[![n8n](https://img.shields.io/badge/n8n-Workflow-FF6C37?logo=n8n)](https://n8n.io/)
[![Gemini](https://img.shields.io/badge/AI-Gemini%201.5%20Flash-4285F4?logo=google-gemini)](https://deepmind.google/technologies/gemini/)

補助金や助成金の情報は、膨大な公募要領の中に隠れています。

AI Subsidy Hunterは、最新の公募情報を自動収集し、AI（Gemini）があなたの事業内容と照らし合わせ、「受給可能性」と「申請のメリット」を即座に判定。チャンスを逃さずSlackへ届けます。

## 🌟 このワークフローで解決すること
- **「検索疲れ」からの解放**: J-Net21や自治体サイトを毎日巡回する手間をゼロにします。
- **「難解な公募要領」を3秒で理解**: 数十ページのPDFをAIが読破し、要点を箇条書きで要約。
- **「申請漏れ」の撲滅**: 締め切り間近の案件や、自社に最適な新着案件をプッシュ通知。

## 🛠 主な機能
1. **インテリジェント・クローリング**: RSSや特定サイトから最新の補助金ニュースを自動取得。
2. **AIマッチング判定**: Geminiが「業種・所在地・従業員数」に基づき、対象外の案件を自動フィルタリング。
3. **優先度スコアリング**: 受給金額と難易度から「取り組むべき優先順位」をAIが独自判定。
4. **アクションプラン生成**: 申請に向けて「まず何を準備すべきか」の初手までをAIがアドバイス。

## 🏗 セットアップ方法
1. **n8nへのインポート**: `ai-subsidy-hunter.json` をインポート。
2. **自社プロフィールの設定**: ワークフロー内の変数（またはスプレッドシート）に、自社の業種・地域・資本金等の情報を入力。
3. **APIキーの設定**:
   - Google AI Studio (Gemini API)
   - 通知先（Slack/Notion/Email等）
