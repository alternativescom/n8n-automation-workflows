# 🔒 Security Audit Bot

Googleスプレッドシートの「アカウント台帳」を定期的に読み込み、パスワード変更から90日以上経過しているユーザーを自動抽出。Geminiが毎月異なる「セキュリティ豆知識」を添えた通知文を作成し、監査実施ログを自動で記録するワークフローです。
「IPA SECURITY ACTION ★★（二つ星）」などの情報セキュリティ対策における、運用エビデンスの記録に最適です。

## ✨ 特徴 (Features)
- **🤖 AI-Powered Alerts:** 毎月同じ文面にならず、Geminiが最新のセキュリティ啓発を交えた人間味のあるアラートを生成。
- **📊 Automated Evidence:** 通知の実行結果をスプレッドシートに自動追記し、監査ログ（証跡）として残します。
- **🎯 ゼロ・システム連携:** Active Directory等の複雑な連携は不要。スプレッドシートの台帳だけで完結します。
