# 🛡️ n8n-workflow #24: Security Audit Bot
> **「見えない脅威」を可視化。AIが24時間体制であなたのシステムを監視・診断する。**

[![Security](https://img.shields.io/badge/Audit-Gemini%201.5%20Pro-4285F4?logo=google-gemini)](https://deepmind.google/technologies/gemini/)

「今の設定で本当に安全か？」という不安を、確信に変えます。
Security Audit Botは、各種スキャンツールやログ、設定ファイルをAI（Gemini）が解析。リスクの優先順位を判定し、具体的な対策手順とともにSlackへ通知します。

## 🌟 このワークフローで解決すること
- **「形だけの監査」からの脱却**: 年に一度のチェックではなく、24時間365日の継続的なモニタリングを実現。
- **難解なログ解析の自動化**: 数万行のログや複雑な診断結果をAIが読破し、「要するにどこが危ないか」を瞬時に特定。
- **ISMS/Pマーク運用の形骸化防止**: 証跡の自動収集と記録により、コンプライアンス維持の工数を劇的に削減。

## 🛠 主な機能
1. **インテリジェント・ログスキャン**: クラウドの設定ミスや不審なアクセスをAIが検知・分析。
2. **脆弱性スコアリング**: 見つかった脆弱性が「今すぐ直すべきか」を、自社の文脈に合わせてAIがランク付け。
3. **自動修復アドバイザー**: 専門用語を並べるだけでなく、「どのファイルをどう書き換えるべきか」の初動対応を提示。
4. **定期監査レポート生成**: 経営層や顧客へ提出できるレベルのセキュリティ状況要約を自動作成。

## 🏗 セットアップ方法
1. **n8nへのインポート**: `security-audit-bot.json` をインポート。
2. **スキャン対象の設定**: AWS/GCPの設定、GitHubのリポジトリ、あるいは特定のサーバーログを連携。
3. **APIキーの設定**:
   - Google AI Studio (Gemini API)
   - 各種スキャンツール/プラットフォームの認証情報
