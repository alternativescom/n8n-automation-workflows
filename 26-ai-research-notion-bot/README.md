# 独自リサーチ：RSS×Gemini×Notion自動蓄積ボット

指定したテーマの最新ニュース（RSS）を自動取得し、Geminiに「プロのコンサルタント視点」で深い考察をさせ、Notionのデータベースへ自動蓄積するワークフローです。
単なるニュース要約ではなく、「このニュースをどう顧客への提案に活かすか」「ブログやSNS発信の切り口はどうするか」までをAIに自動生成させることで、コンサルティング業務や情報発信の質とスピードを劇的に高めます。

## ✨ 主な機能

1. **RSS自動取得:** GoogleニュースなどのRSSフィードから最新記事を自動的に検知します。
2. **AIによる高度なインサイト抽出:** Geminiが記事を読み込み、「3行サマリー」「提案アイデア（営業トーク）」「情報発信の切り口」をMarkdown形式で構造化して出力します。
3. **Notionデータベースへの自動蓄積:** 抽出したインサイトを、元の記事URLとともにNotionへ自動保存。自分だけの「最強のネタ帳」が全自動で構築されます。

## 📦 含まれるファイル

* `workflow.json`: n8nにインポートして使用するワークフロー定義ファイルです。

## ⚠️ Notion連携時の重要ポイント（必ずお読みください）

n8nとNotionを連携させる際、以下の設定を忘れると「データベースが見つからない（No results）」というエラーが発生します。

1. **データベースは「フルページ」で作成する:** インライン（ページ内への埋め込み）のテーブルではなく、独立したフルページのデータベースとして作成してください。
2. **「コネクトの追加」を行う:** Notionのデータベース画面右上にある「・・・」メニューから「コネクトの追加（または接続）」を選び、n8nのインテグレーションを**明示的に招待（許可）**してください。

## 🚀 導入手順

1. 本リポジトリの `workflow.json` をn8nにインポートします。
2. `RSS Feed Trigger` ノードを開き、監視したいRSSのURLを設定します。
3. `Gemini: Analyze News` ノードにAPIキーを設定し、必要に応じてプロンプト内の「自身の職業・ターゲット層」をカスタマイズしてください。
4. `Notion: Save to Database` ノードにNotionのAPI設定（内部インテグレーションシークレット）を入力し、保存先のデータベースIDを指定します。
5. 列（プロパティ）の紐づけを行い、テスト実行でNotionに保存されることを確認したらActiveにします。
